# Community Theme Studio for Odoo 18

Community Theme Studio is a powerful, modern UI customization engine built specifically for Odoo 18 Community Edition. It introduces a beautiful, real-time Owl-based floating configuration panel that allows users to seamlessly customize their Odoo workspace without writing a single line of CSS.

## 🚀 Features

- **Real-time Live CSS Engine:** All visual changes are applied instantly in the browser without needing a page refresh or server restart.
- **Dynamic Color Palettes:** Easily change Primary and Secondary theme colors utilizing intuitive native color pickers.
- **Dark Mode Integration:** Full support for Odoo 18's native Bootstrap 5 Dark Theme, including customized `.o_dark_mode` CSS for beautiful high-contrast List, Kanban, Calendar, and Pivot views.
- **Glassmorphism UI:** Add beautiful blurred glass effects to the Odoo Navbar with a single toggle.
- **Background Image Upload:** Personalize your workspace by uploading custom background imagery, complete with an adjustable overlay opacity slider.
- **Custom Favicon Upload:** Instantly update your browser's tab icon dynamically.
- **One-Click Presets:** Switch between hand-crafted themes instantly (Ocean, Forest, Cyberpunk, Minimal).
- **Persistent Settings:** All configurations are persistently saved to the Odoo backend (`ir.config_parameter`) and apply instantly on page load.

## 🛠️ Installation

1. Clone or download this repository into your Odoo 18 `custom_addons` directory.
2. Restart your Odoo server.
3. Log in as an Administrator, activate Developer Mode, and click **Update Apps List**.
4. Search for `Community Theme Studio` and click **Activate**.

## 💻 Usage

Once installed, simply open your user profile menu in the top right corner of the Odoo navbar and select **Theme Studio**. The floating panel will slide in from the right, allowing you to tweak and preview your UI in real-time. Click **Save** to persist your changes globally!
